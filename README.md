# CraftCMS Baukasten

## What's recommended
- Laravel Valet (https://laravel.com/docs/8.x/valet)
- PHP 7.4
- MySQL 8 / MariaDB
- optional: Redis `brew install redis`

## Configs
- Duplicate the `.env.example` file as `.env` and do your settings
